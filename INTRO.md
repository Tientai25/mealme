# 🍽️ MealMe - Hôm nay ăn gì?

> Web app giúp bạn lên thực đơn thông minh dựa trên mục tiêu và nguyên liệu có sẵn!

![Status](https://img.shields.io/badge/status-completed-success)
![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Tính năng chính

🎯 **Mục tiêu cá nhân** - Giảm cân, tăng cân, giữ dáng, healthy, tiết kiệm

🔍 **Gợi ý thông minh** - Nhập nguyên liệu có sẵn, nhận gợi ý món ăn phù hợp

📅 **Thực đơn tự động** - Tạo menu 1 ngày hoặc 7 ngày không trùng lặp

❤️ **Lưu yêu thích** - Bookmark món ăn bạn thích

---

## 🛠️ Tech Stack

**Frontend:** React 18 + Vite + CSS

**Backend:** Node.js + Express + MongoDB

**Auth:** JWT + bcryptjs

---

## 🚀 Quick Start

```bash
# Backend
cd backend
npm install
npm run seed
npm run dev

# Frontend (terminal mới)
cd frontend
npm install
npm run dev
```

**→ Mở:** http://localhost:3000
